Folder 02_trimming contains:
- adapters.txt - list of adapters to be trimmed
- fastp_trimming.sh - script file for fastp trimming
- trimming.sh - script file for trimmomatic trimming
- overrrepresented.xlsx - file containing OR sequences for each library
- cutadapt/ - directory for cutadapt attempts
- fastp/ - directory for fastp attempts
- trimmomatic/ - directory for trimmomatic attempts

every subdirectory contains txt.files with explanations of every attempt; adapters used and the fastqc files 
of the trimmed files.
