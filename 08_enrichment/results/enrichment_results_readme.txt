Data:
- panther.xlsx - results form the online Panther GO enrichment analysis
- go_results_bp_de_simplified.txt - simplified, BP results using clusterprofiler()
- go_results_cc_de_simplified.txt - simplified, CC results using clusterprofiler()
- go_results_mf_de_simplified.txt - simplified, MF results using clusterprofiler()
- go_annotations.xlsx - GO annotations, gene classification based on GO distribution at a specific level.
- go_annotation_bp.txt - GO annotations, BP
- go_annotation_cc.txt - GO annotations, CC
- go_annotation_mf.txt - GO annotations, MF
- goresults_mf.txt - GO over-representation analysis results, MF, for the 241 DE genes
- goresults_bp.txt - GO over-representation analysis results, BP, for the 241 DE genes
- goresults_cc.txt - GO over-representation analysis results, CC, for the 241 DE genes
- go_annotation_mf.txt - GO annotations, MF
- annotation_de.txt - GO annotations, only for the DE genes
- goresults_bp_de_1369.txt - GO enrichment results, for BP, for the 1369 DE genes
- goresults_cc_de_1369.txt - GO enrichment results, for CC, for the 1369 DE genes
- goresults_mf_de_1369.txt - GO enrichment results, for MF, for the 1369 DE genes

- kegg.txt - KEGG pathway over-representation analysis results
- kegg_module.txt - KEGG module over-representation analysis results
